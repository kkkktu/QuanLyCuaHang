# BiluxuryProject_Model_WithMain

Bản này đã có file main chạy chương trình:

```text
src/GUI/Main.java
```

Main class cần chạy:

```text
GUI.Main
```

Cấu trúc package:

```text
src/GUI
src/model
src/database
```

Lưu ý: thư mục `model` phải viết thường, không dùng `Model`.

## Cần thêm

- Copy JDBC driver vào `lib`
- Copy ảnh vào `src/image` nếu cần: `logo.png`, `admin.png`, `nhanvien.png`
- Chạy SQL trong `sql/BiluxuryDB_full_seed.sql`
